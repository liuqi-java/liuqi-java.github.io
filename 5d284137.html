网站信息

language 和主题相关 Next 7.8.0 简体中文 使用的 zh-CN 官方文档上是说使用zh-Hans 是以前的版本使用的

部署到 GitHub

到 GitHub 新建一个 repository。如果你希望你的站点能通过 <你的 GitHub 用户名>.github.io 域名访问，你的 repository 应该直接命名为 <你的 GitHub 用户名>.github.io

清除缓存，打包，部署
